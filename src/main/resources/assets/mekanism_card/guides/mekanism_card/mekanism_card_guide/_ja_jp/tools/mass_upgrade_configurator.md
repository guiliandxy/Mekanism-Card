# 一括アップグレード設定器

<ItemImage id="mekanism_card:mass_upgrade_configurator" />

一括アップグレード設定器は、複数の Mekanism 機械に対してアップグレードを一括でインストールまたは削除できる強力なツールです。

<RecipeFor id="mekanism_card:mass_upgrade_configurator" />

## アップグレードの選択

ツールを使用する前に、インベントリまたはバインドされた AE2/QIO ネットワークにアップグレードモジュールを用意しておく必要があります。

ツールはインベントリ、AE2、QIO の順にアップグレードモジュールを検出します。ツールチップにはインベントリで選択されているアップグレード種別が表示されます。

## ネットワークストレージ

一括アップグレード設定器は、バインドされた AE2 ネットワークまたは QIO 周波数からアップグレードモジュールを取り出せます。

- AE2：Wireless Access Point のリンクスロットでバインドします。
- QIO：周波数が選択された QIO ブロックをスニーク + 右クリックしてバインドします。
- 消費優先度：インベントリ、AE2、QIO。

## モード

設定器には2つのモードがあります：

- インストールモード：機械にアップグレードをインストールします。
- 削除モード：機械からアップグレードを削除します。

空中で右クリックするとモードを切り替えます。

## 半径モード

半径モードでは：

- スニーク + 機械を右クリック：隣接するすべての機械に操作を適用します。
- 空中で右クリック：インストール/削除モードを切り替えます。
- スニーク + 空中で右クリック：選択モードを切り替えます。

## 選択モード

選択モードでは：

- スニーク + 右クリック：選択範囲の1つ目または2つ目の角を設定します。
- 右クリック：選択範囲内のすべての機械に一括操作を実行します。

2つの角を選択して直方体の範囲を定義し、機械を右クリックして実行します。

## 中クリックショートカット

- 機械を中クリック：対応するすべてのアップグレードモジュールを自動で取り付けて満杯にします。
- 半径モードで中クリック：隣接するすべての機械に一括インストールします。
- 選択モードで中クリック：選択範囲内のすべての機械に一括インストールします。
